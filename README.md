# FUTURE\_DS\_01 - Business Sales Dashboard





### 📌 Task Overview



###### This task involves analyzing e-commerce sales data to identify key business insights such as sales trends, top-performing products, customer demand patterns, and overall revenue performance.

###### ✔ Sales by Category
